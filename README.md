# PLANDLY_PROJECT
일정관리 및 동기부여를 할 수 있는 웹사이트, PLANDLY.



### 1. 프로젝트 정보
- JSP 서블릿과 톰캣서버, 마이바티스를 활용하여 만든 일정관리 웹사이트입니다.
- 개발기간 : 2023.09.04 ~ 2023.09.19


### 2. 개발환경
- IDE : Eclipse
- Database : Oracle DB
- Framework : Apache Maven
- ORM : MyBatis


## 주요기능
- 로그인 및 회원가입


    카카오 로그인 API를 활용한 간편로그인 가능, 회원가입 시 아이디 중복체크 확인 및 DB값에 정보 저장.


- 인생카운트 시계
